ModMonster's Game Repository

Prison Break

Prison Break is a short game that I made with Bitsy where you try to escape a prison! It's pretty simple, and there's not even a way to lose!

Lander

Lander is a game where you try to land a UFO on a landing pad. You need to make sure that you land in the center of the pad, as well as making sure that you don't hit it too hard.

Flappy Cave

Flappy Cave is like Flappy Bird, except you are flapping in a cave instead of outside. This makes the game much easier! It's also the first game I made in PICO-8, so don't expect too much!
