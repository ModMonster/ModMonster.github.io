About Code Generator

Code Generator is a program to encrypt and decrypt messages using an encryption key.
When you encrypt a message, you get back a scrambled string that doesn't resemble the original text at all. If you give the encrypted string to someone, as well as the encryption key that you used, they can put it into the decoder to decrypt the message and get back the original string that you typed.
